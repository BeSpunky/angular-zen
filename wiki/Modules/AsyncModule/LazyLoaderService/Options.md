# LazyLoaderService Options
Both `loadScript()` and `loadStyle()` have a default behaviour and only require a url to work.

* See [`ScriptLoadOptions`](https://dev.azure.com/BeSpunky/BeSpunky%20Libraries/_git/angular-zen?path=%2Fprojects%2Fbespunky%2Fangular-zen%2Fasync%2FLazyLoader%2Fscript-load-options.ts&version=GBmaster) for `loadScript()` API.
* See [`StyleLoadOptions`](https://dev.azure.com/BeSpunky/BeSpunky%20Libraries/_git/angular-zen?path=%2Fprojects%2Fbespunky%2Fangular-zen%2Fasync%2FLazyLoader%2Fstyle-load-options.ts&version=GBmaster) for `loadStyle()` API.
